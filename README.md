# 🖼️ StegoShield – Secure Image Steganography System

## 📌 Project Overview
StegoShield is a secure image steganography system designed to conceal sensitive information within digital images using the Least Significant Bit (LSB) technique. The project focuses on enhancing data privacy by enabling covert communication while preserving the visual integrity of images.

---

## 🎯 Objective
The primary objective of this project is to develop a reliable and efficient method for hiding confidential information in images without raising suspicion, ensuring both security and data integrity.

---

## 🛠 Technologies Used
- **Programming Language:** Python  
- **GUI Framework:** Tkinter  
- **Techniques:** LSB Steganography, Bit Manipulation  
- **Data Structures:** Bloom Filter, Trie  

---

## ⚙️ Methodology
- Implemented **LSB (Least Significant Bit)** technique to embed secret data into image pixels  
- Used **bit-level operations** for accurate encoding and decoding  
- Integrated **Bloom Filter** for efficient duplicate detection  
- Applied **Trie data structure** for optimized message validation  
- Ensured minimal distortion to maintain original image quality  

---

## ✨ Key Features
- Secure encoding of hidden messages within images  
- Accurate decoding and retrieval of embedded data  
- Support for PNG and JPEG image formats  
- Graphical User Interface for ease of use  
- Preservation of image quality after data embedding
